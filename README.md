# Calculadora de Salário

## Sobre a Atividade

Esta aplicação foi desenvolvida como uma atividade prática de programação, com o objetivo de auxiliar os alunos na aplicação de conceitos fundamentais, tais como:

* Entrada e saída de dados;
* Declaração e utilização de variáveis;
* Operações matemáticas;
* Manipulação de valores monetários;
* Organização da lógica de programação.

## Objetivo da Lição

Coletar dados do usuário (salário e percentual de descontos) para realizar os seguintes cálculos:

* Valor líquido após os descontos;
* Valor total anual;
* Valor do 13º salário;
* Valor de férias.

## Competências Desenvolvidas

Ao concluir esta atividade, o aluno será capaz de:

* Receber dados informados pelo usuário;
* Realizar cálculos matemáticos utilizando variáveis;
* Exibir resultados formatados;
* Compreender a aplicação da programação em situações reais do cotidiano.

## Funcionalidades

✅ Cálculo do salário líquido após descontos

✅ Cálculo do valor anual recebido

✅ Cálculo do 13º salário

✅ Cálculo do valor de férias

---

## Demonstração

O programa solicita ao usuário o salário mensal e o percentual de descontos. A partir dessas informações, realiza automaticamente os cálculos e apresenta os resultados na tela.

### Exemplo de Execução

**Entrada**

* Salário Mensal: R$ 3.000,00
* Percentual de Desconto: 10%

**Saída**

* Valor Líquido: R$ 2.700,00
* Valor Total Anual: R$ 32.400,00
* Valor do 13º Salário: R$ 2.700,00
* Valor de Férias: R$ 3.600,00

## Como Executar

### Executável

O arquivo executável (.exe) encontra-se na pasta:

```text
executavel/
```

Para executar o programa, certifique-se de possuir o Java instalado e execute:

```bash
java -jar CalculadoraSalario.jar
```

### Código-Fonte

Os arquivos-fonte do projeto podem ser acessados e editados através do Eclipse na pasta:

```text
src/
```

Nessa pasta encontram-se todas as classes Java utilizadas no desenvolvimento da aplicação.

Este projeto foi desenvolvido exclusivamente para fins didáticos, servindo como exercício prático para reforçar conceitos de lógica de programação, estruturas básicas da linguagem Java, entrada e saída de dados, cálculos matemáticos e organização de código.
